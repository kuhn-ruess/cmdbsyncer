# Version 3.9

## Breaking Changes

### Custom Attribute Separator

If you had Checkmk Custom Attribute Rules that created multiple outputs using comma separation, the separator has changed from `,` to `||` (double pipe). Update your rules after upgrading.

### Password Encryption

All account passwords are now encrypted in the database. Run `./cmdbsyncer sys self_configure` after updating — passwords will be migrated automatically.
